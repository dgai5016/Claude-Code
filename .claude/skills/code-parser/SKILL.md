我是**前端与AI领域的初学者**，请你用最简单、最通俗、最细致的方式帮我解析下面这段源码。
我正在学习 Claude-Code 源码，需要你严格遵守以下规则：

--------------------------------------------------
【1】文件整体解释
- 放在【代码最顶部】
- 用初学者能听懂的话说明：文件作用、模块定位、调用关系、设计意图
--------------------------------------------------

【2】代码块解释
- 每个重要代码块【上方】写一段自然语言解释
- 用大白话讲：这段代码干什么、为什么这么写、核心逻辑
- 避免专业术语堆砌，尽量通俗
--------------------------------------------------

【3】行级别注释
- 关键行、难懂行、核心逻辑
- 在【代码行正上方】添加注释
- 不修改原代码
- 不删除原注释
--------------------------------------------------

【4】绝对禁止
- 不修改原代码任何字符
- 不删除原注释
- 不改变结构
- 不重构
- 不使用过于高深的专业术语
--------------------------------------------------

【5】输出格式
1. 顶部文件说明（初学者友好版）
2. 代码块上方 → 通俗块解释
3. 关键行上方 → 简单易懂注释
4. 输出可直接保存为 .ts 文件
